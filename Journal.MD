# 7/21/2026 5 AM - Analyzing the Spotify Display Project

_Time spent: 2.0h_

In this session, I spent 2 hours understanding the workflow of the whole project and planning about the materials. In the original tutorial,
I had to design the case, an enclosure to support the display and the ESP32. However, I decided to instead work on the project to make it 
like a circuit. Basically, the same concept but no enclosure. I planned on prototyping my hardware since I wouldn't have to solder anything
and I can just use a breadboard, a solderles device. First, I planned on my materials by deciding on an ESP32 Microcontroller, 1.8" TFT 
Display, male-to-male jumper wires, three small breadboard-friendly push buttons, and two breadboards. Then, I planned on the workflow for
my project, MacBook -> USB-C connector/interface -> ESP32 Microcontroller and tiny buttons mounted on the breadboard
(Button 1 - previous song; Button 2 - play/pause; Button 3 - next song) -> Jumper Wires connecting from the breadboard to the TFT Display
(vital for SPI Communication), which will be mounted on another breadboard. Then, I planned on how I would wire my components since the 
original tutorial doesn't tell me how to wire my components. Then, I had nothing to plan much since the connection to the Spotify API and the
software was provided in the original tutorial.
